📊 Sales Performance Dashboard (Excel)

Project Overview

This project analyzes sales data using Excel to identify trends, regional performance, and product insights. An interactive dashboard is created to visualize key business metrics.

🎯 Objectives

Analyze sales performance across regions

Identify top-performing products and categories

Track monthly sales trends

Evaluate delivery performance

🧰 Tools Used

Microsoft Excel

Pivot Tables

Charts

Slicers

📊 Key Features

Interactive dashboard with filters (Region, Category, Segment)

KPIs: Total Sales, Total Profit, Total Orders, Avg Delivery Time

Sales trend analysis (monthly)

Category and regional performance analysis

📈 Insights

West region shows highest sales performance

Technology category contributes significantly to revenue

Sales peak observed in November

Delivery time averages around 4 days
